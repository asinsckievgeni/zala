Генератор плейлиста Zala

Этот скрипт позволяет создать плейлист IPTV из незашифрованных каналов для сервиса Zala, используя данные из их API.

Идея взята отсюда. Скрипт переделан с помощью ИИ, так как автор не умеет программировать.

Зависимости

Для работы скрипта необходимо установить модуль requests:

pip install requests

Использование

Просто запустите скрипт без аргументов:

python zala.py

Скрипт отправит GET-запрос по ссылке на файл с данными о каналах и получит ответ в формате JSON. Затем он пройдёт по списку каналов и добавит в плейлист те, которые не зашифрованы и используют протокол HLS. Плейлист будет сохранён в файле zala.m3u в той же папке, где находится скрипт. Скрипт также выведет сообщение об успешном создании плейлиста.
